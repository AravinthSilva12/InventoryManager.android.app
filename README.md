Inventory Manager android mobile app 
Stock management, Bill processing, Supplier relationship, CRM, all in one place aimed in this app.
App architecture : Bottom nav bar -> Stock | Supplier | Home | Bill | CRM
Each tabs holds distinct responsibilities and interconnected to eachother.
Model View ViewModel (MVVM) architecture followed.
Room database connected for stock items to store the stock data locally on any device for the user.
Currently project is in progress and not yet completed.
