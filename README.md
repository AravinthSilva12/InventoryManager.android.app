Project Name - Inventory Manager android mobile app. 
Description - This project deals with building an android app that helps with Stock management, Supplier data, Customer data and Billing activities.
App is for whom? Retail business people (Stores, small shops)
Features - Stock management, Bill processing, Supplier relationship, CRM, all in one place aimed in this app.
App architecture - Bottom nav bar -> Stock | Supplier | Home | Bill | CRM
Each tabs holds distinct responsibilities and interconnected to eachother.
Model View ViewModel (MVVM) architecture followed.
Room database connected for stock items to store the stock data locally on any device for the user.
Project status - Project is completed with Minumum viable product features.
